# Telegram Sender

## Setup
- Get required information:
    - Create a bot:
        - Chat @BotFather in Telegram.
        - Type “/newbot” and follow the prompts to create a new bot.
    - Get chat id:
        - Chat @get_id_bot in Telegram.
- Create `.env` file
- Fill these parameters inside the `.env`:
```
TOKEN="xxxx"
CHAT_ID="xxxx"
```
